# Ipl_data_Analysis
🏏 IPL Data Analysis (2008–2025)

An interactive IPL (Indian Premier League) Data Analysis Dashboard built to explore team performance, player statistics, and season insights from 2008 to 2025.

📊 Project Overview

This project provides a comprehensive analysis of IPL seasons using data visualization techniques. It highlights key metrics such as:

Season Winners & Runner-ups
Total 6’s, 4’s, Teams, Venues
Orange Cap & Purple Cap statistics
Most Runs, Wickets, Fours, Sixes
Team-wise Points Table

The dashboard allows users to dynamically select seasons and explore insights.

🚀 Features

✅ Interactive Season Filter (Dropdown)
✅ Dynamic KPI Cards (Runs, Wickets, Boundaries)
✅ Player Performance Insights
✅ Team Rankings & Points Table
✅ Clean and Professional UI Design
